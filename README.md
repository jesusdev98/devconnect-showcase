# 🚀 DevConnect

A full-stack social platform for developers focused on scalability, interaction systems and testing.

---

## 🧠 Overview

DevConnect is a real-world application built with a modern full stack:

- Frontend → Angular (SPA)
- Backend → Laravel (REST API)
- Authentication → Laravel Sanctum (session-based)
- Database → MySQL (production) / SQLite (testing)
- Environment → Docker
